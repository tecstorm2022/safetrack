What is Safetrack?

Application for monitoring vital signs

Safetrack

Safetrack is a project created in Unity during the national Hackaton Tecstorm 2022 for the category Digital Health by the Safe4Health team from ciTechCare IPL.

Safetrack is a monitoring system for patients after they receive medical discharge by the professional.

The patients will receive a sensor that will be applied after they receive discharge, permitting the app to monitor their vital signals.

In case a signal reaches irregular values the application is going to alert the professional, and save it to the patient history, that way the professional can analyze the problem and evaluate the patientâ€™s state.

The vital signs will be displayed in a simple form in real time, the professional can also view the parameters in a 3D augmented reality with the use of a QR Code.

The code is openly available on Github under the MIT License

Features

Safetrack features the following features:

Patient Managment

Safetrack helps the professional to keep track of their patients, they can perform various actions and interact with them.

Real time vital signs

Safetrack shows in real time the data read from sensors attached to the patient, sending alerts in case of any irregular values.

Patient History

Safetrack saves every alert generated by the sensors so that the professional can evaluate and study the patient's case.

Augmented Reality

Safetrack as an AR feature, with it the user can scan a QR code and see the affected organ in 3D and the values it is generating.
